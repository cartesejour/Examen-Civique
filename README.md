# 🇫🇷 Simulateur Examen Civique Français 2026

![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![License](https://img.shields.io/badge/License-Copyright-red)

Une application web moderne et fluide conçue pour aider les candidats à préparer l'entretien civique pour la **Carte de Séjour (A2)** et la **Carte de Résident (B1)**.

## 🚀 Fonctionnalités
- **Niveaux au choix** : Sélection entre le test CSP ou CR.
- **Traductions d'aide** : Support multi-langues (Anglais, Arabe, Russe, etc.) pour les questions et les réponses.
- **Joker 50/50** : Possibilité d'éliminer deux mauvaises réponses après une courte publicité simulée.
- **Correction Intégrale** : Accès à l'analyse détaillée des 40 questions après le test.
- **Sauvegarde locale** : Affichage du dernier score obtenu à l'accueil.

## 🛠️ Structure du Projet
Le projet est architecturé de manière modulaire pour faciliter la maintenance :
- `index.html` : Structure de l'application.
- `style.css` : Styles personnalisés et animations.
- `js/` : Logique fragmentée (API, Quiz Engine, UI, App).
- `data/` : Base de données des questions au format JSON.
